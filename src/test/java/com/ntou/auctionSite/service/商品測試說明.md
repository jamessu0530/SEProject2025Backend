# ProductServiceTest 類別說明

此 Java 測試類別使用 Spring Boot 測試框架對 `ProductService` 進行單元測試，主要功能包括：

## 功能概述

1. **列出所有商品**
   - 呼叫 `productService.getAllProduct()`
   - 印出商品名稱、ID、價格與狀態

2. **測試新增商品**
   - `testCreateProduct()` 建立新的 `Product`
   - 驗證新增後的商品 ID、名稱、價格是否正確

3. **測試商品上架**
   - `testPublishProduct()` 將商品狀態設為 `ACTIVE`
   - 驗證商品狀態是否成功更新

4. **透過 ID 取得商品**
   - `testRetrieveProductById()` 透過商品 ID 取得商品
   - 驗證商品存在並印出名稱與價格

5. **取得所有商品**
   - `testGetAllProducts()` 呼叫 `printAllProducts()`
   - 驗證商品清單非空且至少有一筆商品

6. **測試編輯商品**
   - `testEditProduct()` 修改商品名稱與價格
   - 驗證修改後的資料是否正確

7. **測試下架商品**
   - `testWithdrawProduct()` 將商品狀態設為 `INACTIVE`
   - 驗證商品狀態是否成功更新

8. **測試刪除商品**
   - `testDeleteProduct()` 刪除指定商品
   - 驗證商品已不存在
