# BidServiceTest 類別說明

此 Java 測試類別使用 Spring Boot 測試框架對 `BidService` 進行單元測試，主要功能包括：

## 功能概述

1. **取得所有拍賣商品**
    - 呼叫 `bidService.getAllAuctionProduct()`
    - 將所有商品的名稱、ID、價格與類型印出

2. **建立與測試拍賣出價流程**
    - 設定一個測試商品 ID 為 `P105` 將他的productType改成AUCTION
    - 使用 `bidService.createAuction()` 建立一個拍賣商品，起標價 100，結束時間為 10 秒後
    - 印出建立的商品資訊與拍賣截止時間

3. **測試出價操作**
    - 透過 `bidService.placeBid()` 嘗試出價高於起標價或高於目前最高價
    - `printBidInfo()` 輔助方法用來印出目前最高價與最高出價者
    - 故意嘗試出價低於起標價，捕捉 `IllegalArgumentException`，出現錯誤訊息是合乎預期的

4. **拍賣結束處理**
    - 使用 `Thread.sleep()` 模擬拍賣時間流逝
    - 呼叫 `bidService.terminateAuction()` 結束拍賣
    - 印出最後得標者與成交價格
